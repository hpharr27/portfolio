Core4 Concept Technical Definition Real-Life Analogy Python Example Scratch Example
Variables A variable is a value that can change based on conditions. My grade can change depending on how well I do on the tests. If you type left a certain set of text will appear and if you type right then a different set of text will appear. Making a score variable that could change if the player hits the ball against the paddle.
Loops A loop is a sequence of instructions that is repeated until a certain condition is met. I'm going to continue to run until I cross the finish line. In python you can loop through a list. In scratch you can drag a "repeat __ times" where you can input a specific number of times you want an instruction to repeat or a forever loop that never ends.
Conditionals
Functions An instuction that will only occur if another instruction happens. I will only get a salad if Katie gets a salad. Only if you type "left" or "right" will each story play its next set of text. Scratch offers "if" "else" statements that are instructions with certain conditions.
